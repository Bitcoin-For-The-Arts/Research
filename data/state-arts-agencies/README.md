# State arts agencies

Datasets covering US state arts agency appropriations, grant rolls, and program-level expenditure data.

## Primary sources

- **NASAA — Legislative Appropriations Annual Survey** — https://nasaa-arts.org/research/funding/legislative-appropriations-annual-survey/
- **Arts Midwest — Public Funding for the Arts in the Midwest** — https://artsmidwest.org/resources/research-data/
- Individual state arts agency annual reports and adopted budgets
